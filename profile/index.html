About me

I am a Graduate student of Master's in Computer Science at William Jessup University, with a focus on full-stack development. Through both academic coursework and hands-on projects, I have developed a strong foundation in a variety of technologies, including React, Python, Django, Java, and SQL/NoSQL databases. My technical skills are complemented by my ability to tackle complex problems, collaborate effectively with teams, and manage time efficiently.

One of my most exciting projects includes a Full-Stack E-Commerce Application, where I developed an end-to-end platform using React for the front end and Node.js for the back end. I implemented features such as user authentication, product management, and a shopping cart to provide a seamless user experience. I’ve also applied machine learning techniques to create a Predictive Model for housing price analysis and built a Real-Time Event Detection System using deep learning to improve security monitoring in CCTV-surveilled areas.

I am continuously expanding my knowledge through certifications, including Python Programming, CSS and HTML essential training and Certified Web Developer (CIW), and I am committed to leveraging my skills to develop impactful software solutions. With a passion for coding, problem-solving, and innovation, I am eager to contribute to projects that push the boundaries of technology and create real-world impact.
